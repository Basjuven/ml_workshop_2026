# Data Analysis with Python

Before talking about *Machine Learning*, we first need to understand something fundamental:

> Artificial intelligence does not start with algorithms… it starts with data.

On this first day, we will learn how to use **Python as a scientific tool** to observe, explore, and ask questions about information.

You do not need previous programming experience.  
Here we are not going to memorize code; we are going to learn how to **think like scientists using data**.

## What will we learn today?

During this session, you will:

- Learn how to import libraries
- Use libraries such as NumPy, Pandas, Matplotlib, and Seaborn
- Understand what a *dataset* is
- Read real data using Python
- Explore information and find patterns
- Perform simple calculations with data
- Create plots to visualize results

---

## Why is this important?

Machine Learning algorithms do not “guess.”

They do exactly what you will do manually today:

1. Observe data
2. Detect patterns
3. Make decisions based on those patterns

Today, you will be the algorithm.

---

:::{tip}
During the workshop, avoid thinking:

> *What does this code mean?*

Instead, ask yourself:

> *What are the data telling me?*

Python will only be our laboratory notebook.
:::

---
